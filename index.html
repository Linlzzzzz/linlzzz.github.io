JAVA学习Day2

2020-03-30

前排建议使用：notepad++文本编辑器
（别问，问就是好用）

标识符注意！！！：
标识符严格区分大小写
标识符不可以数字开头，不可以单独使用关键字或保留字（但是可以包含）
标识符可以是英文字母，数字 , _ , $

java中变量的定义：数据类型 变量名 = 变量的值
例如：int i = 1;
变量的作用域在一对{}之间有效
注意和C的区别： java中 int i ； 是错误的 因为没有给变量赋值

八种基本数据类型：
数值型（整数类型【byte，short，int，long】，浮点类型【float，double】）
字符型（char） 布尔型（boolean）

类型 占用存储空间 范 围
byte 1字节=8bit -128127
short 2字节 -2^152^15-1
int 4字节 -2^312^31-1
long 8字节 -2^632^63-1

示例：byte b =126
int i = 1
short s =1
long a = 3l (不要忘记加 L )

单精度float 4字节（7位精度）
双精度double 8字节（16位精度）
例： float f = 1.22f
double b = 1.22

字符：用英文的单引号括起来的单个字母，数字，符号
例如： char c1 = ‘a’ ;
char c2 = ‘ ' ‘ ; <—表示定义单引号，“\”为转义字符

转义字符 说明
\b 退格符
\n 换行符
\r 回车符
\t 制表符
" 双引号
' 单引号
\ 反斜线

boolean布尔类型： true 和 false (不是1和0！！！)
例如： boolean b1 = true

注：除了八种基本数据类型其余的都是引用数据类型

引用数据类型：
类（class）<—string字符串在这里 接口（interface） 数组（ [ ] ）

字符串：由0到多个数字字母符号共同组成的一个串，这个串要用英文的双引号括起来
String str = “hello kunkun”;
System.out.println(str);

凡是引用类型都可以用null作为值，也就意味着可以在初始化的时候赋值为null
String是引用类型，也就是说可以用null作为值

若 String s0 = “hello”
String s1 = “hello”
则只能由一个hello。假设hello的内存地址xxxx，声明s0变量时给s0赋值实际上让s0变量引用hello的内存地址xxxx
当再声明变量s1也赋值hello时,实际上也是直接把已经存在的hello的内存地址给s1引用
也可以这样写： String s3 = “hel” + “lo”

关于数据类型自动转换：小容量可以赋值给大容量 反过来不行
byte,char,short—>int—>long—>float—>double
注：byte,char,short之间不会相互转换,他们三者在计算时首先转换为int类型

当把任何类型的值和字符串进行连接运算时(+),基本类型的值将自动转化为字符串类型。

关于转换问题：
byte b = 9;
int i = b;
以上这两行属于正常隐式转换数据类型（自动的）
int k = 7;
byte b0 = (byte)k;
以上属于强制转换数据类型，转换的数据类型用英文的小括号括起来（手动的）
字符串类型转换后期再总结
布尔类型不可以转换为其他数据类型

关于算术运算符：
加减法与四则运算不在多讲
自增（前）：先运算后取值 例如：a = 2; b = ++a; 结果是a=3;b=3
自增（后）：先取值后运算 例如：a = 2; b = a++; 结果是a=3;b=2
自减同理（i++先用后加，++i先加后用）

关于%模运算：
System.out.println(5%-2);的值为1，就是说%右边带负号和不带一样
System.out.println(-5%2);的值为-1，%左边的正负才影响最后的值

Hexo使用日记

2020-03-25

第一次搭建博客成功，总结下搭建博客的经验。
首先搭建博客需要安装必要的软件，比如 git 。 （详细请参考官网） 其次在搭建博客的过程中，
_config.yml 文件里的内容参数尤为重要。 windows 系统用户记得在每个标题后的冒号处的后方
多打一个空格，不然程序会报错。在hexo d操作中输入账号密码的那一步，密码是不显示的，但是
不妨碍用户输入，正常操作即可。
电脑中创建好个人博客后要与github里的个人仓库相连接，把个人博客存入仓库中就可以方便的在
云端看到自己的博客。也可以在_config.yml文件更改个人主题，主题可从百度下载，记得克隆。
以后的日子里我的个人博客会不定时更新。内容什么都有，还希望大家多多访问。
